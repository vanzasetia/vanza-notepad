# Vanza Notepad

Vanza Notepad is a part of Tiny Project made by me, Vanza Setia.

## What are the benefits?

- No ads.
- No trackers.
- No proprietary JavaScript.
- Zero bloat.
- Written in HTML, CSS, and JavaScript.
- Hackable.
- Use Grammarly browser extension easily.
- Built to last for _at least_ 30 years.
- Free web app.
